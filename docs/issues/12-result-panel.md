# Issue #12: 結果パネル（ResultPanel）

### 背景 / 目的
ルート検索結果（距離・時間・経由情報）を表示するコンポーネントを実装する。ユーザーがルート情報を一目で確認できるようにする。

- 依存: #10, #11
- ラベル: `frontend`, `ui`

### スコープ / 作業項目
- [ ] コンポーネント作成
  - `src/components/SidePanel/ResultPanel.tsx`
    - 距離表示（例: 32.4 km）
    - 所要時間表示（例: 45分）
    - ルート概要（例: 首都高速経由）
  - `src/components/SidePanel/RouteInfo.tsx`（距離・時間表示）
  - `src/components/SidePanel/StepList.tsx`（ターンバイターン案内、任意）
- [ ] ローディング状態の表示
  - スピナーまたは「検索中...」テキスト
- [ ] 結果なし状態の表示
  - プレースホルダーまたは非表示
- [ ] ダークモードUI適用

- [ ] 距離（例: 32.4 km）が表示される
